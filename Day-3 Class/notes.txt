


CSS  :
------
    ->CSS stands for Cascading Style Sheet used to apply stylings  like colorings , layouts, animations  , responsive designing..etc
    ->CSS 1994  1996  www
    ->present version is CSS 3 

Method apply CSS web page : 
---------------------------
    1. inline CSS / inline Stylings 
        ->applying stylings by taking style Attribute in html ELEMENTs
        ->here not required CSS selectors



    2. internal CSS / internal Stylings 
        ->applying stylings by taking style tag in head part 
        ->here required CSS selectors

    3. External CSS / external Stylings 
        ->applying stylings by taking external new CSS file 
        ->here we need to link CSS to HTML by taking  <link> in head section 
        ->here required CSS selectors

CSS selectors : 
----------------
    ->CSS selectors are used to target / select HTML ELEMENTS 
    ->we can use this css selectors in internal / external CSS
